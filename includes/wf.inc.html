WritingFantasy

WritingFantasy is a Twine story format designed by Sophie Houlden, with a similar aim to Gordian Book â€” outputting a Twine story as a gamebook. Gordian Book and WritingFantasy have many bits of shared functionality, but WF aims to produce an HTML output suitable for EPub, while GB is focussed on PDF output and export. You can use WritingFantasy as a proofing tool for Gordian stories, if you like, or any other workflow that makes sense.

Compatibility

Gordian Book implements WritingFantasy passage tags as synonyms for its own to allow a story to work in both formats, as follows:

  • hidden is treated as a synonym for skip
  • ending is treated as a synonym for last
  • fixednumber_1 is treated as a synonym for 1, to prenumber a passage
  • frontmatter is processed the same
  • backmatter is processed the same
